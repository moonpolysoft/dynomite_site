Dynomite

Dynomite is a distributed key value storage system. It is designed from the ground up to support a worst case scenario for most database systems: highly concurrent low latency random reads and writes. Dynomite achieves this goal by borrowing many of the design elements from Amazon's Dynamo paper.

Dynomite provides a means for almost any language to get at your data. There is a thrift interface which can be accessed from any of Thrift's supported languages. There is also a text based protocol designed to be easily parseable from just about any language, with a reference client written in ruby.
